## Candy delivery app
REST service written on Flask for "Yandex Backend School"

## Installation
```
python3 -m pip install poetry --user
git clone https://github.com/kiselz/candy-delivery-app.git
cd ./candy-delivery-app
poetry install
```

## Running
Development
```
make run-dev
```
Production
```
make run-prod
```
Test
```
make run-test
```

## Running all test
Just write
```
make test
```
